# Fady Morcos Portfolio

Single-page portfolio rebuilt with Vite, React, Tailwind CSS, Framer Motion, and Swiper.

## Stack

- Vite + React
- Tailwind CSS
- Framer Motion
- Swiper
- Vitest + Testing Library

## Scripts

- `npm run dev` - start local dev server
- `npm run build` - create production build in `dist/`
- `npm run preview` - preview built app locally
- `npm run test` - run tests once
- `npm run test:watch` - run tests in watch mode

## Content Model

All portfolio content is centralized in:

- `src/data/portfolioData.js`

Update this file to change hero text, client links, experience roles, app details, about copy, and contact links without editing section components.
